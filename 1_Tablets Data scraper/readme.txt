This project is a simple "web scraping script" built with Python. It scrapes product details (name, price, description, and reviews) of tablets from a sample e-commerce test site and saves the data into a CSV file.

=>Features
->Extracts "product names"  
->Extracts "prices"  
->Extracts "descriptions"  
->Extracts "review counts"  
->Stores data in a structured "CSV file" (`Tablets Data.csv`)

=>Technologies Used

->Python  
->Requests , For fetching the webpage  
->BeautifulSoup ,For parsing HTML content  
->Pandas ,For creating and saving data in CSV format
