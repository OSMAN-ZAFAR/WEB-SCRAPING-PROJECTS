PriceOye Mobile Scraper

This project is a simple yet effective web scraping script built with Python and BeautifulSoup.  
It automatically scrapes the latest mobile phone details (name, price, discount, and reviews) from **PriceOye.pk** and saves the data into a CSV file for further analysis or reporting.  

=>Features  
-> Scrapes PriceOye mobile listings across multiple pages.  
-> Extracts:
  -> Mobile Name  
  -> Price  
  -> Discount (if available)  
  -> User Reviews  
-> Handles missing values gracefully.  
-> Stores all data in a CSV file (`sample_output.csv`).  
-> Easy to customize for other e-commerce websites.  

=>Tech Stack  
-> Python 3.8+  
-> Requests  
-> BeautifulSoup (bs4)  
-> Pandas  
-> lxml
